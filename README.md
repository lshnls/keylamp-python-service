# Сервис управления переключением цвета лампы при изменении раскладки клавиатуры

## Описание

`keylamp‑python‑service` — Python‑сервис для запуска и контроля скрипта `keylamp.py` как фонового сервиса.
Подходит для автоматического старта, установки и поддержки работы Keylamp‑логики.

## Структура проекта

- `keylamp.py` — основной файл сервиса  
- `install.sh` — скрипт установки  
- `README.md` — документация  

## Установка и запуск

```bash
git clone https://github.com/lshnls/keylamp-python-service.git
cd keylamp-python-service
sh install.sh
```
